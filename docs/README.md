# FlightHack Documentation

This repository contains documentation for the FlightHack project.

## Overview

FlightHack is a web application that helps users find the best deals on flights. It provides users with a simple and intuitive interface to search for flights based on their preferences and budget.

## Features

- Search for flights by departure and destination airports, departure and return dates, and other criteria.
- View detailed information about available flights, including prices, airlines, and departure/arrival times.
- Save favorite flights and view booking options.
- User authentication and account management.
- Integration with external flight data providers to ensure up-to-date and accurate flight information.

## Getting Started

To get started with FlightHack, follow these steps:

1. Clone the repository: `git clone https://github.com/your-username/FlightHack.git`
2. Install dependencies:
   - Frontend: `cd frontend && npm install`
   - Backend: `cd backend && npm install`
3. Start the development servers:
   - Frontend: `cd frontend && npm start`
   - Backend: `cd backend && npm start`
4. Access the app in your web browser at `http://localhost:3000`.
